# AiAgentSkills
Various skills for AI agents
To deploy skills to AI agent, navigate to the agent directory under home. for example ~/.claude. Make a directory call skills if it doesnt exist. Then make a directly under the skills and name it whichever skill name you want, for example, C++_WinRT_AppSDK. then copy the skill.md into this. Reload your agaent, and run /skills in the prompt to check if the skill is loaded.

Directory structure:
~/.claude
  |-Skills
      |-C++_Winrt_appsdk
           |-- skill.md
